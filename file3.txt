	- above request accepts the JSON Object
	  as "Request Body".

		{
	
		   "id":6,

		   "p_name":"product_six",
	
		   "p_cost":600000

	- in general we will use PUT Request for
	  updation purpose.

	- above request accepts the JSON Object as
	  POST Parameter.

		{

		   "p_name":"p_six",
	
		   "p_cost":60000
